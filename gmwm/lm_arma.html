lm_arma. gmwm 2.9.1

MLR in Armadillo

Usage

lm_arma(y, X)

Arguments

y
A vec of length $N x 1$ containing the responses.
X
A mat with dimensions $N x p$, which is the design matrix.

Value

A field<vec> with:
coef
Coefficients

resid
Residuals

sigma2
Sigma^2

Description

Perform Multiple Linear Regression using armadillo in C++

Examples

x = cbind(1,1:10)
y = cumsum(rep(.45,10))

lm_arma(y, x)[[1]]

[1,] -2.042667e-16
[2,]  4.500000e-01
