install_data_package. gmwm 2.9.1

Install a SMAC Group Data Package

Usage

install_data_package(pkg = "imudata", type = "ONL", loc = NULL)

Arguments

pkg
A string with the package name to install.
type
A string with value "ONL" or "LOCAL"
loc
A string that contains the file location.

Description

Downloads and Install the a SMAC Group Data Package to use.

Details

For a list of data packages please see the SMAC-Group data repo http://smac-group.com/datarepo.

Examples

## Not run: ------------------------------------
# # Online install
# install_data_package("imudata")
# 
# # Local install
# install_data_package("LOCAL","C:/Users/James/Documents/imudata_x.y.z.tar.gz")
