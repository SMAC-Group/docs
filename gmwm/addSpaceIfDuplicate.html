addSpaceIfDuplicate. gmwm 2.9.1

Add Space to Avoid Duplicate Elements

Usage

addSpaceIfDuplicate(x)

Arguments

x
A character vector

Description

Add space to every element if there are duplicates in the vector.

Examples

##no duplicate
x1 = c('GMWM2', 'GMWM1', 'GMWM3')
addSpaceIfDuplicate(x1)


[1] "GMWM2" "GMWM1" "GMWM3"




##duplicate
x2 = c('GMWM3', 'GMWM4', 'GMWM3', 'GMWM4', 'GMWM5', 'GMWM6','GMWM3')
addSpaceIfDuplicate(x2)


[1] "GMWM3"   "GMWM4"   "GMWM3 "  "GMWM4 "  "GMWM5"   "GMWM6"   "GMWM3  "
